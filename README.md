hlblock.list - компонент для системы Bitrix Framework. Позволяет выводить записи highload блоков в публичной части.

- поддержка полей сортировки
- поддержка постраничной навигации
- поддержка пользовательской фильтрации выборки
- возможность выбора полей для отображения
- кеширование

Необходимые модули:
- highloadblock
